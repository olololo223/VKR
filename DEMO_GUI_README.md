# GUI Демо приложение для распознавания эмоций

## Описание

Windows Forms приложение на tkinter для распознавания эмоций в реальном времени с использованием камеры и/или микрофона.

## Функциональность

- ✅ Распознавание эмоций в реальном времени
- ✅ Использование камеры (видео)
- ✅ Использование микрофона (аудио)
- ✅ Мультимодальное распознавание (камера + микрофон)
- ✅ Чекбоксы для включения/выключения камеры и микрофона
- ✅ Автоматический поиск обученных моделей

## Требования

- Python 3.8+
- Обученные модели в `logs/training/`
- Камера и/или микрофон

## Установка зависимостей

```bash
pip install tensorflow opencv-python pyaudio librosa numpy pillow
```

## Запуск

```bash
python run_demo.py
```

или

```bash
python src/demo_gui.py
```

## Использование

1. **Запустите приложение** - оно автоматически найдет последние обученные модели
2. **Выберите источники**:
   - ✅ "Использовать камеру" - включено по умолчанию
   - ✅ "Использовать микрофон" - включено по умолчанию
3. **Нажмите "Начать"** - начнется распознавание эмоций
4. **Наблюдайте результаты**:
   - Отображается текущая эмоция
   - Отображается уверенность предсказания
5. **Нажмите "Остановить"** - остановка распознавания

## Режимы работы

- **Только камера** (микрофон выключен): визуальное распознавание
- **Только микрофон** (камера выключена): аудио распознавание
- **Оба источника** (камера + микрофон): мультимодальное распознавание (если модель загружена)
- **Оба выключены**: ничего не происходит (предупреждение)

## Эмоции

Приложение распознает 7 эмоций:
- Злость (angry)
- Радость (happy)
- Страх (fear)
- Грусть (sad)
- Отвращение (disgust)
- Удивление (surprise)
- Нейтрально (neutral)

## Примечания

- Приложение автоматически ищет последние обученные модели в `logs/training/`
- Если мультимодальная модель не найдена, используется комбинация визуальной и аудио моделей
- Для лучшей точности рекомендуется использовать оба источника одновременно
